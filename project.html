Projects

  • Data Pipeline

    This data pipeline reads data provided from TriMet C-TRAN. Once the data is gathered, it is then filtered as it sends it through a buffer called Apache Kafka. Once the data has gone through the buffer, it is then sent to a PostgreSQL table. Being in a PostgreSQL table allows for the user to create exportable data reports. These files allow for data maps to be made that represent the query. This project uses Python, Linux bash scripts, Apache Kafka, PostgreSQL, and MapboxGL.
